Automated Deployment Strategies in Kubernetes

About

This is a simple webpage created as an example for implementing and comparing various deployment strategies in Kubernetes.

Description

The deployment strategies covered are:
1. Canary
2. Blue-Green
3. Rolling updates

Goals

  • Configuring each deployment strategy to understand the behaviour, benefits and trade-offs.
  • Providing practical insights for choosing and implementing the appropriated deployment strategy in real-world scenarios.
